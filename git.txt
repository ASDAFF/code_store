Добавление коммита:
- git status (проверка статуса)
- git add . (добавление в коммит всех файлов)
- git commit -m "название комита" (называем комит, желатьно по шаблону "#номер_задачи нескольких словах суть сделаных правок")
- git push (отправляем коммит)

для пенереоса коммита на другую ветку (Merge Request), например хотим перенести на dev
- по ssh переходим на ветку dev
- git status проверяем статус, чтобы не было не закомиченных файлов
- заходим на http://git.o2k.ru
- делаем Merge Request
- git pull - на ветку dev получаем изменения


для получения изменений с мастера на свою ветку:
находясь на своей ветке делаем: git pull origin master
